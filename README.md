# 📊 Power BI — Análisis de Utilidad y Márgenes | Utility & Margin Analysis
### Appol | 2019–2021


## 💡 Business Insights

Estos son los hallazgos clave del análisis, en lenguaje de negocio:

**1. Los ingresos cayeron pero los gastos no**
Entre 2019 y 2021, los ingresos de Appol muestran una caída sostenida mientras los gastos se mantienen estables. El resultado: el margen se erosiona año a año sin que sea obvio en una tabla plana.

**2. PC concentra el 39% de los ingresos totales**
Con $1.316M sobre $3.36M totales, PC es el producto que sostiene el negocio. Si esa línea cae, el impacto es desproporcionado respecto al resto del catálogo.

**3. América genera el 39% de la utilidad global**
Es el continente más rentable por lejos. Asia y Europa están lejos. Cualquier decisión de expansión debería partir de entender qué hace diferente al mercado americano.

**4. El margen global de 19.95% es frágil**
Con ingresos bajando y gastos estables, ese margen no tiene mucho colchón. Una caída adicional en ventas de PC o América lo llevaría a zona de riesgo.

## 🇬🇧 English

### 📌 Project Overview
Interactive Business Intelligence dashboard built with **Microsoft Power BI**, analyzing sales performance, utility, and profit margins for **Appol** across multiple continents and countries during the period **2019–2021**.

### 🎯 Objectives
- Visualize total utility and profit margins by continent, country, and product
- Identify sales trends across quarters and years
- Compare performance across regions: América, Asia, Europa, África, and Oceania

### 📁 Files
| File | Description |
|------|-------------|
| `PROYECTO_FINAL_FedericoOscarGiglio.pbix` | Power BI dashboard file |
| `Appol Datos.xlsx` | Source dataset used for the analysis |

### 📊 Dashboard Pages
1. **Portada** — Cover / Landing page
2. **Indicadores Margen** — Margin KPIs
3. **Indicadores Tiempo** — Time-based indicators
4. **Indicadores Productos** — Product performance
5. **Indicadores Gastos-Ingresos** — Expenses vs Revenue

### 📈 Key Metrics
- **Total Utility:** $671.13 million
- **Total Margin:** 19.95%
- **Top continent by utility:** América (39.14%)
- **Period covered:** 2019–2021

### 🛠️ Tools & Technologies
- Microsoft Power BI Desktop
- Microsoft Excel (.xlsx)
- DAX (Data Analysis Expressions)
- Power Query (ETL)

### 🗂️ Data Model
The model includes the following tables:
- `Ventas 1` & `Ventas 2` — Sales transactions
- `Continentes` — Continent dimension
- `Países` — Country dimension
- `Productos` — Product dimension
- `Calendario_auto` & `Calendario_manu` — Date tables
- `_Medidas` — DAX measures table
